# Intelligent Warehouse

Simple Java console application for warehouse/product management with SQLite.

## Status

This project is still under development.
Some features are implemented, some are partial, and some menu options are still placeholders.

## Current Scope

- view warehouse data
- add products
- delete products
- update product name
- update product price
- increase and decrease quantity
- sort products by price
- export products to TXT
- show cheapest and most expensive product
- count total warehouse value

## Tech Stack

- Java
- SQLite
- IntelliJ IDEA

## Running

1. Open the project in IntelliJ IDEA.
2. Make sure the SQLite database path in the source code matches your machine.
3. Compile and run `Main.java`.

## Notes

- The project uses the local SQLite file, you can create Db by `scheam.sql`.
- Some menu options are not finished yet.
